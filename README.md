# テーマ

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公キャラを十字キーで動かし、シフトキーで攻撃してスコアを競うゲーム

## ゲームの実装
* 背景、主人公と、敵の描画
* 敵を倒すことで、スコアを獲得
* 自身が倒されるまでのスコアを競い、20000点でクリア
* スコアや倒した敵の数に応じて、ゲームの難易度が上昇する
* スコアや倒した敵の数に応じて、各攻撃が強くされる
* 回復アイテムを拾うと、体力が回復する
* スコアが2000点以上あれば、キーCを押下で、10秒間敵のリスポーンと速度を遅延させられる
